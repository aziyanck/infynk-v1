The square favicon. Dark green gradient (`#0b2206 → #103607 → #0d2c06`) circle (radius 233.21) with the same stacked-cards logo cut out in `#f2f3f3`. Used as `index.html` favicon.

## Used By

## Risks

* SVG gradients are not always rendered by email clients / some Android browsers; modern browsers handle them fine.
* The file name contains no spaces — note the second variant `logo favicon-nb.svg` *does* contain a space.
